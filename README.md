# Ale Tracker

Track Allegro parcels directly from terminal.

## Usage
```shell
./ale_tracker.py ALE000000001111 ALE000000002222
+-----------------+--------------------+----------------------------------------------+
| Parcel          | Date               | Last status                                  |
+-----------------+--------------------+----------------------------------------------+
| ALE000000001111 | 13 Dec 2022, 12:31 | Kurier przekazał przesyłkę do magazynu       |
| ALE000000002222 | 14 Dec 2022, 21:37 | Przesyłka została przygotowana przez nadawcę |
+-----------------+--------------------+----------------------------------------------+
```
